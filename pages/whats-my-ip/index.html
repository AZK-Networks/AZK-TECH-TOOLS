How to Check if You Have CGNAT
  • Open your router's admin page and navigate to the WAN section
  • Note down the public IP address listed in the WAN section
  • Compare the IP address from step 2 with the IP address displayed on this page
  • If they match, you are not under CGNAT. If they differ, you are likely under CGNAT.
